This project consist of using Webpack, JavaScript modules and OOP to build a todo list app.


## Technologies used

- HTML
- CSS
- Bootstrap
- JavaScript
- Webpack
- ESlint
- Prettier
- Stickler-ci
- Git
- IDE (WebStorm)

### Install
- Clone the app
- `npm install`

### Development server
- `npm start`

### Production build
- `npm build`
